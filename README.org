* The Step
1. ~mkdir -p     PROJECT_NAME~
   ~chmod -R 755 PROJECT_NAME~
   ~cd           PROJECT_NAME~
2. ~cabal init~
3. Edit the ~.cabal~ file
   - add dependencies
   - modifey other options
4. Write the code
5. ~cabal run~
